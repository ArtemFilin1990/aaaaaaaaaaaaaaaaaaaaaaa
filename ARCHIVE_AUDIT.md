# Аудит архивов

## Что сделано
- Распакованы **все архивы** из корня репозитория (`.zip`, `.rar`) с рекурсивной обработкой вложенных архивов.
- Для каждого распакованного файла выполнен машинный просмотр содержимого:
  - для текстовых форматов сохранён предпросмотр первых строк;
  - для бинарных/крупных файлов сохранён контрольный SHA-256 (первые 1 MiB).
- Результаты сведены в индекс `ARCHIVE_CONTENT_INDEX.md`.

## Структура раскладки
- Распакованные данные помещены в каталог `unpacked_archives/`.
- Каждый архив распаковывается в отдельную директорию формата:
  - `unpacked_archives/NN__<имя_архива>/`
- Внутри каждой директории сохраняется `_extract.log` с логом извлечения.

## Где смотреть содержимое
1. Полный перечень архивов и файлов с предпросмотром: `ARCHIVE_CONTENT_INDEX.md`.
2. Сами распакованные данные: `unpacked_archives/`.
3. Скрипт для повторного прогона: `scripts/unpack_archives.py`.

## Воспроизводимость
Повторить процедуру можно командой:

```bash
python scripts/unpack_archives.py
```

Скрипт пересобирает `unpacked_archives/` и заново генерирует `ARCHIVE_CONTENT_INDEX.md`.
