# Внутренний порядок ревью перед правкой

1. Открыть корень проекта.
2. Найти package.json / pyproject.toml / requirements.txt / Makefile.
3. Найти entrypoint.
4. Найти слои: schema, route, handler, service, UI.
5. Зафиксировать, что сломается рядом.
6. После правки прогнать доступные проверки.
7. Перед ответом убрать воду.
