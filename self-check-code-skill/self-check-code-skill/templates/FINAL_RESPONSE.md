# Короткий формат ответа

```text
Сделано:
- что поменял
- где поменял

Проверил:
- какие команды реально запустил
- что прошло / что не прошло

Риск:
- один главный риск
- или: не вижу
```

## Пример

```text
Сделано:
- вынес интеграцию Checko в отдельный сервис
- добавил разделы Налоги и Закупки в меню бота

Проверил:
- npm run build — прошло
- node --check server/routes.js — прошло
- сценарий reply-меню проверен по коду

Риск:
- live-проверка API не прогнана без реальных ключей
```
