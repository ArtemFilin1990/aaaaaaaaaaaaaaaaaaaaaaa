# Пример запроса

Исправь callback меню в Telegram-боте, добавь раздел Налоги и проверь, что сборка проходит.

## Как должен работать скилл

- сначала читает package.json и роуты;
- потом правит нужные файлы;
- потом гоняет build и синтаксис;
- в ответе пишет только: что сделал, что проверил, где риск.
