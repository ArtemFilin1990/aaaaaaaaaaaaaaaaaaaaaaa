# Patch: Dadata + Checko для Replit

## Что добавлено
- поле `checkoApiKey` в конфиг бота;
- поле ввода Checko API key в панели `Dashboard`;
- reply-меню Telegram после поиска по ИНН;
- разделы:
  - Реквизиты
  - Финансы
  - Налоги
  - Закупки
  - Проверки
  - Арбитраж
  - Исп. производства
  - Лицензии
  - Риски
- формат финансов в рублях;
- вывод `ГИР БО ID` и официальных ссылок на отчёты ФНС, если они приходят из Checko;
- вывод `СведУплГод` и `СумНедоим` в разделе `Налоги`;
- вывод `ДатаИсп` в разделе `Закупки`, если она приходит из Checko.

## Что нужно сделать на Replit
1. Заменить файлы проектом из этого архива.
2. Выполнить:
   - `npm install`
   - `npm run db:push`
   - `npm run dev`
3. В панели бота заполнить:
   - Telegram token
   - Dadata API key
   - Dadata secret key (опционально)
   - Checko API key (опционально, но нужен для расширенных разделов)

## Важно
- Если `CHECKO API key` не задан, бот всё равно ищет по ИНН через Dadata и показывает краткую карточку.
- Разделы `Финансы`, `Налоги`, `Закупки`, `Проверки`, `Арбитраж`, `Исп. производства`, `Лицензии`, `Риски` требуют Checko key.
- Текущая версия делает упор на ЮЛ/ИП. Отдельное ветвление меню под `person` оставлено на следующий этап.
